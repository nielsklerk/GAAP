Due to the expansion of the universe, the light emitted by galaxies gets stretched, cosmological redshift.

\begin{equation}
    z = \frac{\Delta\lambda}{\lambda_\mathrm{emitted}}=\frac{\lambda_\mathrm{observed}-\lambda_\mathrm{emitted}}{\lambda_\mathrm{emitted}}
\end{equation}
So redshift measurements are important to study the large-scale structures and the evolution of the universe. The most accurate measurements of the redshift are spectroscopic redshifts. This involves using spectroscopy to find the redshift directly, for example, by looking at specific atomic lines or spectral features. However, there is a problem with this method. It can only be applied to a handful of objects at a time. Thus, this method takes a considerably longer time to find the redshift for a large number of sources. Photometry can be used to find the redshift for a large number of sources at the same time. Photometry is, in essence, spectroscopy with a really low resolution.
